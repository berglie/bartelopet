## Color Scheme

**From tailwind.config.ts:**

- **Primary (Brown):** #8c7355 - Used for primary buttons, links
- **Accent (Green):** #42896f - Used for success states, Vipps elements
- **Destructive (Red):** Error messages and destructive actions
- **Muted:** Secondary text, backgrounds

**Component Colors:**

- Email icon background: `primary-100` (light brown)
- Vipps icon background: `accent-100` (light green)
- Success badge: `accent-500` (green)
- Error background: `destructive/10` (light red)
- Card backgrounds: `muted/50` (very light gray)

---

## Responsive Behavior

### Mobile (< 768px):
- Full width buttons
- Modal fills most of screen
- Text remains readable
- Touch targets are adequate (44px+)

### Tablet (768px - 1024px):
- Cards at comfortable width
- Modal centered with padding
- Two-column layouts where appropriate

### Desktop (> 1024px):
- Max width containers (max-w-md, max-w-lg)
- Centered layouts
- Comfortable spacing
- Hover states visible

---

## Animation States

1. **Button Click:** Slight scale/opacity change
2. **Modal Open:** Fade in backdrop + slide in content
3. **Loading Spinner:** Continuous rotation
4. **Success Appear:** Fade in with slight slide
5. **Error Shake:** Subtle shake animation (could be added)
